# omnifood

This project is the final assignment of the _Build Responsive Real-World Websites with HTML and CSS_ course, by Jonas Schmedtmann.

The goal was to build a landing page for a fictional startup called Omnifood, a company that uses artificial intelligence to create and deliver personalized healthy meal plans.
The website was built using only HTML5 and CSS3, applying concepts learned throughout the course.

What the project includes:

- Hero section
- Product features
- Meal examples
- Pricing table
- Testimonials
- Call-to-action section
- Footer

Technologies used:
HTML5
CSS3 (Flexbox, Grid, Responsive Design)

Purpose:
This project consolidates the fundamentals of front-end development and simulates a real-world landing page build.
